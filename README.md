**install python requirements**
***
`pip3 install -r requirements.txt`

**go to the directory**
****
`cd ./frontendapp`

`npm install`

**go back to the project root**

`cd ..`
****
`python3 manage.py makemigrations`

`python3 manage.py migrate`

**run backend**

`python manage.py runserver`



**go to the frontend directory**
****
`cd ./frontendapp`

**run frontend**

`npm run serve`
